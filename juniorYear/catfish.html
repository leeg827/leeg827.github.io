CATFISH CATFISH
PROJECT 3: INTENTIONALLY UNINTENTIONAL

A book made using a element of randomness when generating content. The content of this book was made using a random number generator to select different catfishes from the MTV show Catfish to document within the book. The book also contains elements of "catfishness" such as upside down text, pixelated faces, and fake profiles to help push the feeling of uncovering deceit, which is a huge part of the show.
